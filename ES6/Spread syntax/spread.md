#### 关键词 
可遍历属性

-------------

#### 描述
```
对象的扩展运算符（...）用于取出参数对象的所有可遍历属性，拷贝到当前对象之中。
```

```javascript
let z = { a: 3, b: 4 };
let n = { ...z };
n // { a: 3, b: 4 }
```

解析：参数对象是```{ a: 3, b: 4 }```，所有可遍历属性 ```a和b```，当前对象即```n指向的对象```，所以n = { a: 3, b: 4 }；
